Thermometer Gauge > Overview

Thermometer gauge, as the name implies is used to display temperatures. The fill level of thermometer indicates the data that you wish to depict.

The thermometer chart is a real-time gauge, which can update it's data at every specified interval, without requiring any page refreshes.

A few outputs of FusionWidgets thermometer gauge can be reproduced as under:

The thermometer gauge does NOT need a pre-defined color range. Instead, it takes in the fill color as an attribute and uses that color to fill up the thermometer.

To create a FusionWidgets thermometer gauge, you need to:

  • Decide the lower and upper limit of the chart
  • Decide the visual properties of the chart
  • Decide the chart value along with the fill color code
